# 💡 Ideas Pool

> 채택 전 아이디어를 카테고리별로 관리하는 공간.
> 채택된 아이디어는 `docs/adr/`로 승격한다.

## Status Labels

| Label | Meaning |
|-------|---------|
| `💡 draft` | 초기 아이디어, 논의 필요 |
| `🔍 reviewing` | 검토/설계 진행 중 |
| `✅ accepted` | 채택 → ADR로 승격 예정 |
| `❌ rejected` | 폐기 (사유 기록 후 보존) |

## Categories

| Directory | Scope |
|-----------|-------|
| `scoring/` | Guardian 등급, 점수 체계, 텔레메트리, 게이미피케이션 |
| `immunity/` | 면역 시스템, antibody, validator, self-healing |
| `ux/` | CLI UX, 출력 포맷, 사용자 경험 |
| `integration/` | MCP, 외부 도구 연동, 에코시스템 |
